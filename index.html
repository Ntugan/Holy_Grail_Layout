The header does not have a position element assigned, so it defaults to static.
The nav bar uses fixed positioning, which position is based upon the viewport, allowing it to stay in the same position even after scrolling.
This article uses relative positioning, which positions this element based upon its parent div.
The aside uses absolute positioning, which is depedent on the webpage itself, setting the element to a specific place on the webpage.
The footer is like the nav bar, set to fixed positioning which allows it to always stay in the same place on the viewport.
